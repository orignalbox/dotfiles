 DWM — window manager (Super = Win key)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

 Focus & Stack
  Super + j/k          focus next/prev window
  Super + Shift + j/k  move window up/down in stack
  Super + h/l          resize master area (shrink/grow)
  Super + i            increase number of master windows
  Super + d            decrease number of master windows
  Super + Return       spawn terminal (st)
  Super + Shift + z    zoom (swap focused with master)

 Layouts
  Super + t            tile layout (default)
  Super + f            floating layout
  Super + m            monocle layout (fullscreen stack)
  Super + Space        toggle between last two layouts
  Super + Shift + Space toggle floating for focused window

 Tags (workspaces)
  Super + 1-9          view tag
  Super + Shift + 1-9  move window to tag
  Super + Ctrl + 1-9   toggle tag view
  Super + 0            view all tags
  Super + Tab          toggle last tag

 Gaps (vanitygaps)
  Super + Alt + u      increase all gaps
  Super + Alt + Shift + u  decrease all gaps
  Super + Alt + 0      toggle gaps on/off
  Super + Alt + Shift + 0  reset gaps to default

 Scratchpad
  Super + `            toggle scratchpad terminal
  Super + Ctrl + `     assign window to scratchpad
  Super + Shift + `    remove scratchpad

 Multi-monitor
  Super + ,/.          focus prev/next monitor
  Super + Shift + ,/.  move window to prev/next monitor

 Bar
  Super + b            toggle bar visibility

 System
  Super + Shift + c    kill focused window
  Super + Shift + q    kill focused window (also in sxhkd)
  Super + Shift + r    restart dwm (selfrestart patch)

 SXHKD — hotkey daemon
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

 Launchers
  Super + Return       terminal (st)
  Super + d            dmenu launcher
  Super + Shift + v    clipboard history (clipmenu)
  Super + e            file manager (thunar)
  Super + Escape       reload sxhkd config

 Screenshots
  Print                full screen → ~/screenshots/
  Super + Print        selection → ~/screenshots/
  Super + Shift + Print  selection → clipboard only

 Volume
  XF86AudioRaiseVolume  volume +5% (pamixer)
  XF86AudioLowerVolume  volume -5% (pamixer)
  XF86AudioMute         toggle mute

 Brightness
  XF86MonBrightnessUp    brightness +10%
  XF86MonBrightnessDown  brightness -10%

 System
  XF86Calculator       galculator
  Super + Shift + l    lock screen (slock)
  Super + Shift + q    kill focused window

 ST — terminal shortcuts
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

 Clipboard
  Ctrl + Shift + C     copy to clipboard
  Ctrl + Shift + V     paste from clipboard
  Shift + Insert       paste from primary selection

 Scrollback
  Shift + PageUp       scroll up
  Shift + PageDown     scroll down
  Shift + MouseWheel   scroll up/down

 Font size
  Ctrl + Shift + PgUp  zoom in
  Ctrl + Shift + PgDn  zoom out
  Ctrl + Shift + Home  reset zoom

 Opacity
  Ctrl + Shift + O     increase opacity
  Ctrl + Shift + P     decrease opacity

 URL copy
  Alt + l              cycle through URLs on screen → copy to clipboard

 Files

  ll      ls -lh (long format)

  install sudo pacman -S
  remove  sudo pacman -Rns
  search  pacman -Ss
