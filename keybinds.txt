DWM Keybindings
(Super is the Mod key)
---------------------------------

Window Management:
  Super + j / k           : Focus window in stack
  Super + Shift + j / k   : Move window in stack
  Super + h / l           : Change master/stack split ratio
  Super + i / d           : Increase/decrease number of master windows
  Super + Return          : Move focused window to master (zoom)
  Super + Shift + c       : Close focused window
  Super + b               : Toggle bar visibility
  Super + Shift + Space   : Toggle window floating
  Super + t               : Set tiling layout
  Super + f               : Set floating layout
  Super + m               : Set monocle layout

Gaps Control (Super + Alt):
  Super + Alt + u         : Increase/decrease all gaps
  Super + Alt + Shift + u : Increase/decrease all gaps (inverted)
  Super + Alt + i         : Increase/decrease inner gaps
  Super + Alt + Shift + i : Increase/decrease inner gaps (inverted)
  Super + Alt + o         : Increase/decrease outer gaps
  Super + Alt + Shift + o : Increase/decrease outer gaps (inverted)
  Super + Alt + 0         : Reset all gaps to default
  Super + Alt + Shift + 0 : Toggle gaps on/off

Tag Management:
  Super + [1-9]           : View tag [1-9]
  Super + Shift + [1-9]   : Move focused window to tag [1-9]
  Super + 0               : View all tags
  Super + Shift + 0       : Make focused window visible on all tags
  
Monitor Control:
  Super + , / .           : Focus previous/next monitor
  Super + Shift + , / .   : Move focused window to previous/next monitor

DWM Session:
  Super + p               : Launch dmenu (application launcher)
  Super + Shift + Return  : Launch terminal (st)
  Super + Shift + r       : Restart dwm in place
  Super + Shift + q       : Quit dwm

Scratchpad:
  Super + grave           : Toggle scratchpad terminal

---------------------------------
System Hotkeys (sxhkd)
---------------------------------

  Print Screen            : Take a screenshot
  Calculator key          : Launch galculator
  Volume keys             : Control master volume (up, down, mute)
  Brightness keys         : Control screen brightness (up, down)

---------------------------------
Zsh Vi-Mode Keybindings
(Enabled by 'bindkey -v' in .zshrc)
---------------------------------

Insert Mode (default):
  Type commands normally.

Normal Mode (Press Escape from Insert Mode):
  h, j, k, l        : Move cursor left, down, up, right
  w, b              : Move word forward, backward
  e                 : Move to end of word
  ^, $              : Move to beginning, end of line
  fx                : Find character 'x' on current line
  tx                : Move to character before 'x' on current line
  gg, G             : Go to first, last line of history
  /text             : Search history forward for 'text'
  ?text             : Search history backward for 'text'
  n, N              : Repeat last search forward, backward
  dd                : Delete entire line
  dw                : Delete word
  d$, d^            : Delete to end, beginning of line
  yy                : Yank (copy) line
  yw                : Yank (copy) word
  p                 : Paste after cursor
  P                 : Paste before cursor
  u                 : Undo last change
  Ctrl+r            : Redo
  v                 : Visual mode (select text)
  i, a, I, A        : Enter insert mode (at cursor, after cursor, beginning of line, end of line)
  R                 : Replace mode (overwrite characters)
  .                 : Repeat last command
